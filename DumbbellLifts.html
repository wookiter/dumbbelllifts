The project

This project is Coursera Practical Machine Learning course project.

The goal of the project is to predict the manner in which people did the exercise - dumbbell lift. Dataset consists of many variables and “classe” variable to predict.

The tasks are to: - build the model - explain used cross validation - calculate out of sample error - explain the choices made - predict 20 different cases from testing dataset

Data is available thanks to http://groupware.les.inf.puc-rio.br/har

Creating training and validation set from training set

I chose random forests as they work good with many variables. As random forests sometimes overfits the data I decided to split training set into training and validation to check chosen model on it before using with test dataset. I decided to use a few models and decide later which to choose for test dataset.
